# Jadwala - Website

**Jadwala** (جدولة) is a comprehensive scheduling system designed to streamline bookings and administrative tasks with real-time data synchronization.

## 🚀 Technologies Used
- **Framework:** React 18 + Vite
- **Language:** TypeScript
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Backend/Auth:** Supabase
- **Styling:** Tailwind CSS + shadcn/ui
- **Icons:** Lucide React

## ⚙️ Installation
1. `git clone https://github.com/amar-eisa/jadwala-w.git`
2. `npm install`
3. `npm run dev`

## 📦 Docker Support
A `docker-compose.yml` and `Dockerfile` are provided:
`docker-compose up --build`
